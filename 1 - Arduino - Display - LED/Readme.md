# Arduino & LED

## Deskripsi Proyek
Pada proyek ini akan dipelajari dan didemonstrasikan cara mengendalikan (drive) nyala komponen light emitting diode atau LED menggunakan mikrokontroler yang tertanam pada board Arduino.
